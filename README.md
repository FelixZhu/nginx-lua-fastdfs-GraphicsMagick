nginx-lua-fastdfs-GraphicsMagick
==================
FastDFS是一个开源的轻量级分布式文件系统，它解决了大数据量存储和负载均衡等问题。 特别适合以中小文件（建议范围： 4KB 到 500MB ）为载体的在线服务，如相册网站、视频网站等等。
